# 判断一个正整数是否为2的n次方

对于这个问题想必大家脑海里第一个冒出来的想法就是`模运算`吧

这里给大家提供一个使用`位运算`的解法，可以使用`&运算`

```ts
const isPowerOfTwo = (num: number): boolean => {
    return (num & (num - 1)) === 0;
}
```

![isPowerOfTwo图片](https://img1.imgtp.com/2022/07/03/4FPWvMht.png)

这里运用到了二进制的与运算
> `与运算` 只有 1 & 1 为 1，其它全为 0

2^n 为 偶数，减1为奇数，
